Parcel Management System

Features

  • Nearest Branch Finder
  • Route Optimization using Dijkstra's Algorithm
  • Priority-Based Delivery with Priority Queues
  • Scalable for multiple parcels and orders

Technologies Used

  • Programming Language: Python
  • Algorithms: Dijkstra's Algorithm, Graph Traversal
  • Data Structures: Graphs, Priority Queues

How It Works

1. Enter parcel details such as destination, priority, and parcel ID.

2. The system calculates the nearest branch and optimized delivery route.

3. Orders are prioritized using a priority queue, ensuring timely deliveries.
